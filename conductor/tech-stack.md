# Technology Stack

## Core Language & Runtime
- **Google Apps Script (JavaScript/TypeScript)**: The primary runtime environment for interacting with Gmail and Google Workspace services.
- **clasp**: For local development and deployment of Apps Script projects.

## Backend Services & APIs
- **Gmail Service (Google Apps Script)**: For reading, labeling, and organizing emails.
- **OpenAI API or Google Gemini API**: For AI-driven categorization, nested labeling, and digest summarization.
- **Properties Service / Google Drive**: For storing the JSON configuration file and audit logs.

## Frontend & UI
- **HTML Service (Google Apps Script)**: For building custom sidebars and dialogs within the Gmail interface.
- **Material Design CSS**: For styling UI components to match the native Gmail aesthetic.

## Development Tools
- **VS Code**: Recommended IDE for local development.
- **ESLint/Prettier**: For consistent code quality and formatting.
- **Vitest**: For testing business logic and categorization rules.
